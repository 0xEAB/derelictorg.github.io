DerelictGLFW3

DerelictGLFW3 is a dynamic D binding to the 3.x series of the GLFW library. DerelictGLFW3 1.0.x supports supports the 3.0 series of GLFW and 1.1.x supports the 3.1 series. Derelict's SharedLibVersion feature is not currently supported.

The following sections explain how to build and use DerelictGLFW3. See the page Using Derelict for more information.

Using With DUB

The easiest way to use DerelictGLFW3 is to use DUB to manage your project and add it as a dependency in your project configuration file. Those using the JSON format should add the following to the "dependencies" field of dub.json.

            "derelict-glfw3": "~>1.1.0"

Those using the Simple Declarative Language (SDLang) should add the following to dub.sdl.

            dependency "derelict-glfw3" version="~>1.1.0"

Both of the above lines will add support for GLFW 3.1. For GLFW 3.0, use "~>1.0.0" instead. See the documentation on using Derelict packages with DUB-managed projects for more information.

Using Without DUB

It is not necessary to use DUB to manage your project in order to use DerelictGLFW3, but it is still required to build it. To do so, make sure DUB, DMD and git are all available on your system path, then execute the following commands to obtain the source code. The simplest way is to use DUB to fetch the latest version of both DerelictGLFW3 and DerelictUtil (the links lead to the DUB registry page for each project, where you can find the latest version number). See the documentation on using Derelict with projects not managed by DUB for more information.

Loading

To load the GLFW library with DerelictGLFW3, call one of the overloads of the load method on the DerelictGLFW3 instance.


            DerelictGLFW3.load();

This no-argument version will attempt to load the GLFW library using a default set of system-specific library names to look for on the system search path. Overloads accept one or more custom library names to search for. See the general documentation on using Derelict for more information relevant to all shared libraries.
